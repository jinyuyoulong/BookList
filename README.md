# BookList
范金龙的书香门第计划-图书目录
自工业革命之后，全球的生产资料出现爆发式的增长，书籍的出版和发行变的越来越容易。这就造成了市面上流通的图书鱼龙混杂，
基于两个现实的境况：
1. 国人分辨好坏的能力太低，受舆论的影响太大。
2. 儿童没有形成完整的价值观之前，易受其他事物的影响。
因此制作本目录，仅为家庭图书馆营造一个相对理性，干净的读书氛围。此为初心。

**此目录为图书索引，用以指导家庭图书收藏计划，是范金龙的书香门第计划的一部分。**

`注：本文档维护规则如下：`
```
1. 书名，作者。附本页推荐理由定位
2. 推荐理由
```
